One clear block. See it through.

A free AI prompt that refuses to let a goal stay vague. It narrows the work to one finishable block—then builds an offline focus file that keeps the task, the reason, and the next step in front of you while you work.

The prompt · paste into any AI
Run a short setup, then output a finished focus file. Ask ONE question at a time, keep it warm and fast. Q1 — MODE. Offline work → plan to turn the internet off. Needs internet → name the only sites the work requires, and block the rest with your usual blocker. Q2 — WORK + TIME. The task and the block window. Q3 — SHAPE THE GOAL. The header is the outcome; the jobs are the steps. Never restate the checklist. Q4 — THE TRAP (optional). "Anything that tends to pull you off task? I'll remind you. (Or skip.)" Q5 — THE REASON. "Why does finishing this matter to you today? I'll show you this if you drift." Then output the focus file, changing only the config to match…

Now open an AI and paste it into a new chat.
